# US-PAGE-01 : Création et définition d'une Page

* **Statut Backend :** `TO DO`
* **Statut Backoffice :** `TO DO`
* **En tant que** utilisateur (via le Backoffice)
* **Je veux** pouvoir créer une nouvelle page web pour mon site
* **Afin de** définir son titre, sa description et son chemin d'accès (URL).

### Critères d'acceptation :
* La page possède un titre obligatoire (`Title`) et une description optionnelle (`Description`).
* Le chemin (`Path`) doit être unique pour identifier l'URL de la page (ex: `/contact` ou `/blog/article-1`).
* Je peux définir si la page est publiée (`IsPublished`).
