| Statut Backend | Statut Backoffice |

* **En tant que** administrateur
* **Je veux** pouvoir consulter les utilisateurs existants
* **Afin de** pouvoir les administrer (les gérer, vérifier leur statut, etc.).

### Critères d'acceptation :

**1. Affichage**
* La liste complète des utilisateurs est renvoyée de manière paginable (Pagination) pour assurer de bonnes performances même avec un grand nombre d'inscrits.

**2. Outils de recherche**
* Il est possible d'appliquer des filtres sur la recherche d'utilisateurs (ex: par identifiant, par email, par statut actif/inactif).
