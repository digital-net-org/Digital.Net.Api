# US-ADMIN-01 : Création d'un utilisateur

| Statut Backend | Statut Backoffice |
|:--------------:|:-----------------:|
|    `DONE` ✅    |      `TO DO`      |
* **En tant que** administrateur
* **Je veux** pouvoir créer un nouvel utilisateur
* **Afin de** lui donner accès au système.

### Critères d'acceptation :

**1. Informations requises**
* Lors de la création, je dois fournir les informations suivantes pour le nouvel utilisateur :
  * Nom d'utilisateur (Username)
  * Adresse email
  * Mot de passe
  * Login (prénom/nom ou identifiant d'affichage)

**2. Traçabilité**
* Cette action doit générer un événement d'audit indiquant de manière précise : qui a été créé, quand l'action a eu lieu, et par qui l'action a été effectuée.
